# 旭儿导航 · EdgeOne Pages 完整版

基于腾讯云 EdgeOne Pages 构建的全栈个人网站，支持书签导航和博客管理。

## 部署步骤

### 1. 创建 KV 命名空间
- 登录 EdgeOne Pages 控制台
- KV 存储 → 新建命名空间（如 `myblog`）

### 2. 创建 Pages 项目
- 关联 GitHub 仓库
- 分支选择 master
- 构建命令和输出目录留空

### 3. 绑定 KV
- 进入项目 → KV 存储 → 绑定命名空间
- 变量名：`NAV_KV`（必须大写）

### 4. 首次登录
- 访问 `/admin`
- 默认密码：`admin123`

## 功能列表

- 首页：侧边栏 + 博客列表/书签卡片 Tab 切换
- 文章：发布、编辑、删除、置顶、标签、封面图、阅读量
- 书签：添加、编辑、删除、排序、Logo、描述
- 站点设置：标题、副标题、Logo、国内线路按钮
- 暗色模式、响应式布局

## KV 绑定

| 绑定名 | 类型 | 说明 |
|--------|------|------|
| `NAV_KV` | KV 命名空间 | 存储所有数据 |

## 默认密码

- 用户名：`admin`
- 密码：`admin123`
